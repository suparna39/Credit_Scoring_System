# 💳 Credit Scoring System (Soroban Smart Contract)

## 📌 Project Description
<img width="1918" height="947" alt="image" src="https://github.com/user-attachments/assets/9be3f168-05aa-460e-aa93-ed04d058fd1d" />



The Credit Scoring System is a decentralized smart contract built on Stellar's Soroban platform. It enables transparent, tamper-proof credit scoring for users based on their financial behavior and transaction history.

This system removes reliance on centralized credit bureaus and empowers users with ownership of their financial reputation.

---

## ⚙️ What it does

- Creates a decentralized credit profile for each user
- Assigns a base credit score
- Updates credit score dynamically based on actions
- Ensures scores stay within a defined range (300–900)
- Stores all data securely on-chain

---

## ✨ Features

- 🔐 **Decentralized Identity-based Credit Profiles**
- 📊 **Dynamic Credit Score Calculation**
- 📈 **Transaction-based Score Updates**
- 🧱 **Immutable & Transparent Storage**
- ⚡ **Fast and Low-cost execution on Soroban**
- 🧩 **Extensible for DeFi lending, insurance, etc.**

---

## 🧠 How it works

1. User is initialized with a base score (500)
2. Each interaction updates their score:
   - Positive behavior → +score
   - Negative behavior → -score
3. Score is bounded between:
   - Minimum: 300
   - Maximum: 900
4. Transaction count is tracked for credibility
